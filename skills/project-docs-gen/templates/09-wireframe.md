# 와이어프레임 설계서 템플릿

## 의존 문서

- 정보구조도 (`sitemap.md`) — 화면 ID, URL, 메뉴 구조
- 기능명세서 (`features.md`) — 입력 필드, UI 동작

## 사용자에게 확인할 것

- 반응형 대응 범위 (Desktop, Tablet, Mobile)
- 피그마 등 디자인 도구 사용 여부
- 주요 화면 우선순위

## 템플릿

```markdown
# 와이어프레임 설계서

| 항목 | 내용 |
|------|------|
| 프로젝트 | {프로젝트 이름} |
| 문서 버전 | v1.0 |
| 최종 수정일 | {YYYY-MM-DD} |
| 작성자 | {작성자} |
| 상태 | `초안` |

---

## 1. 설계 기준

- **브레이크포인트**: Desktop(1280px+) / Tablet(768px~1279px) / Mobile(~767px)
- **그리드**: Desktop 12컬럼 / Tablet 8컬럼 / Mobile 4컬럼
- **디자인 도구**: {Figma / 없음 (마크다운 설명)}

---

## 2. 공통 레이아웃

### Header

```
┌─────────────────────────────────────────────┐
│ [로고]          [메뉴1] [메뉴2]    [로그인] │
└─────────────────────────────────────────────┘
```

- 로고: 좌측 고정
- GNB: 중앙 또는 우측
- CTA 버튼: 최우측

### Footer

```
┌─────────────────────────────────────────────┐
│ [로고]  [서비스 소개] [이용약관] [개인정보] │
│                          Copyright © 2026   │
└─────────────────────────────────────────────┘
```

## 3. 화면별 와이어프레임

### SCR-AUTH-001: 로그인

**URL:** `/login`

**Desktop:**
```
┌─────────────────────────────────────────────┐
│                   Header                    │
├─────────────────────────────────────────────┤
│                                             │
│           ┌──────────────────┐             │
│           │    [서비스 로고]  │             │
│           │                  │             │
│           │  이메일           │             │
│           │  [___________]   │             │
│           │                  │             │
│           │  비밀번호         │             │
│           │  [___________]   │             │
│           │                  │             │
│           │  [   로그인   ]   │             │
│           │                  │             │
│           │  회원가입 | 찾기  │             │
│           └──────────────────┘             │
│                                             │
├─────────────────────────────────────────────┤
│                   Footer                    │
└─────────────────────────────────────────────┘
```

**UI 컴포넌트:**
| 컴포넌트 | 타입 | 설명 |
|---------|------|------|
| 이메일 입력 | Input[email] | placeholder: "이메일을 입력하세요" |
| 비밀번호 입력 | Input[password] | 토글 show/hide |
| 로그인 버튼 | Button[primary] | 클릭 시 FEAT-AUTH-001 실행 |
| 회원가입 링크 | Link | SCR-AUTH-002로 이동 |
```
