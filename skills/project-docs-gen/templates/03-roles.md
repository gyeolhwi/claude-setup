# 역할/권한 매트릭스 템플릿

## 의존 문서

- 요구사항정의서 (`requirements.md`) — 기능 카테고리, 사용자 유형 참조

## 사용자에게 확인할 것

- 사용자 역할 종류
- 역할별 접근 가능 범위
- 특수 권한 (예: 슈퍼관리자)

## 템플릿

```markdown
# 역할/권한 매트릭스

| 항목 | 내용 |
|------|------|
| 프로젝트 | {프로젝트 이름} |
| 문서 버전 | v1.0 |
| 최종 수정일 | {YYYY-MM-DD} |
| 작성자 | {작성자} |
| 상태 | `초안` |

---

## 1. 역할 정의

| 역할 | 코드 | 설명 | 가입 방식 |
|------|------|------|---------|
| 비회원 | GUEST | 로그인하지 않은 사용자 | - |
| 일반 회원 | USER | 회원가입한 사용자 | 자유 가입 |
| 관리자 | ADMIN | 서비스 관리자 | 내부 지정 |
| {역할} | {코드} | {설명} | {방식} |

---

## 2. 기능별 권한 매트릭스

범례: `O` 허용 / `X` 불가 / `△` 조건부 허용

### {카테고리명} (예: 회원관리)

| 기능 | GUEST | USER | ADMIN | 비고 |
|------|-------|------|-------|------|
| {기능명} | X | O | O | |
| {기능명} | O | O | O | |

### {카테고리명} (예: 콘텐츠)

| 기능 | GUEST | USER | ADMIN | 비고 |
|------|-------|------|-------|------|
| 목록 조회 | O | O | O | |
| 상세 조회 | O | O | O | |
| 작성 | X | O | O | |
| 수정 | X | △ | O | 본인 작성분만 |
| 삭제 | X | △ | O | 본인 작성분만 |

---

## 3. 데이터 접근 범위

| 역할 | 접근 가능 데이터 범위 |
|------|-------------------|
| GUEST | 공개 데이터만 |
| USER | 공개 데이터 + 본인 데이터 |
| ADMIN | 전체 데이터 |

---

## 4. 특수 권한

| 권한명 | 대상 역할 | 설명 |
|--------|---------|------|
| {권한명} | {역할} | {설명} |
```
