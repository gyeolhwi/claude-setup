# 정보구조도 (Sitemap) 템플릿

## 의존 문서

- 요구사항정의서 (`requirements.md`) — 기능 카테고리
- 역할/권한 매트릭스 (`roles.md`) — 페이지 접근 권한

## 사용자에게 확인할 것

- 주요 메뉴 구조
- depth 수준 (보통 2~3 depth)
- GNB / LNB 구분

## 템플릿

```markdown
# 정보구조도 (Sitemap)

| 항목 | 내용 |
|------|------|
| 프로젝트 | {프로젝트 이름} |
| 문서 버전 | v1.0 |
| 최종 수정일 | {YYYY-MM-DD} |
| 작성자 | {작성자} |
| 상태 | `초안` |

---

## 1. 전체 구조 개요

```
{프로젝트명}
├── 공개 영역 (GUEST 접근 가능)
│   ├── 홈 (SCR-HOME-001)
│   ├── 소개 (SCR-INTRO-001)
│   └── 로그인/회원가입
│       ├── 로그인 (SCR-AUTH-001)
│       └── 회원가입 (SCR-AUTH-002)
│
├── 회원 영역 (USER 이상)
│   ├── 마이페이지 (SCR-MY-001)
│   └── {주요 기능}
│       ├── 목록 (SCR-{CAT}-001)
│       ├── 상세 (SCR-{CAT}-002)
│       └── 작성/수정 (SCR-{CAT}-003)
│
└── 관리자 영역 (ADMIN)
    ├── 대시보드 (SCR-ADMIN-001)
    └── {관리 기능}
```

---

## 2. 화면 목록

| ID | 화면명 | URL | 접근 권한 | 설명 |
|----|--------|-----|---------|------|
| SCR-HOME-001 | 홈 | `/` | GUEST+ | 메인 페이지 |
| SCR-AUTH-001 | 로그인 | `/login` | GUEST | 로그인 폼 |
| SCR-AUTH-002 | 회원가입 | `/register` | GUEST | 회원가입 폼 |
| SCR-MY-001 | 마이페이지 | `/my` | USER+ | 개인 정보 |
| SCR-ADMIN-001 | 관리자 대시보드 | `/admin` | ADMIN | 관리 현황 |

---

## 3. 네비게이션 구조

### GNB (Global Navigation Bar)

| 메뉴 | 링크 | 노출 조건 |
|------|------|---------|
| 홈 | `/` | 항상 |
| {메뉴명} | `/{path}` | {조건} |

### LNB (Local Navigation Bar)

| 상위 메뉴 | 하위 메뉴 | 링크 |
|---------|---------|------|
| {상위} | {하위} | `/{path}` |

---

## 4. 공통 컴포넌트

| 컴포넌트 | 노출 위치 | 설명 |
|---------|---------|------|
| Header | 전체 | GNB 포함 |
| Footer | 전체 | 링크, 정보 |
| Sidebar | 관리자 영역 | LNB |
```
