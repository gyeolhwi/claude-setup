## 2. 플로우 상세

### FLOW-AUTH-001: 로그인 플로우

```mermaid
flowchart TD
    A([시작]) --> B[로그인 페이지 접근]
    B --> C[이메일/비밀번호 입력]
    C --> D{유효성 검사}
    D -- 실패 --> E[에러 메시지 표시]
    E --> C
    D -- 성공 --> F[서버 인증 요청]
    F --> G{인증 결과}
    G -- 실패 --> H[로그인 실패 메시지]
    H --> C
    G -- 성공 --> I[토큰 저장]
    I --> J{리다이렉트 URL 존재?}
    J -- 있음 --> K[원래 페이지로 이동]
    J -- 없음 --> L[홈으로 이동]
    K --> M([종료])
    L --> M
```

**분기 조건:**
| 조건 | 결과 |
|------|------|
| 유효성 검사 실패 | 에러 메시지 표시 후 재입력 |
| 5회 이상 실패 | 30분 계정 잠금 |
| 인증 성공 | 이전 URL 또는 홈으로 이동 |

### FLOW-AUTH-002: 회원가입 플로우

```mermaid
flowchart TD
    A([시작]) --> B[회원가입 페이지]
    B --> C[정보 입력]
    C --> D{유효성 검사}
    D -- 실패 --> E[에러 표시]
    E --> C
    D -- 성공 --> F{이메일 중복 확인}
    F -- 중복 --> G[중복 안내]
    G --> C
    F -- 사용 가능 --> H[계정 생성]
    H --> I[환영 이메일 발송]
    I --> J[로그인 페이지로 이동]
    J --> K([종료])
```
